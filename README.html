QuickDock

.NET 8 WPF Windows

轻量级 Windows 快捷启动工具，支持自定义程序快捷方式、天气显示、系统资源监控。

功能特性

快捷启动

自定义添加应用程序、文件夹、网址等快捷方式，一键启动。

天气显示

实时显示当前城市天气状况，支持手动刷新。

资源监控

实时显示 CPU、内存、GPU 利用率。

开机自启

支持设置开机自动启动，静默运行。

热区触发

鼠标移至屏幕指定区域自动显示/隐藏。

系统托盘

最小化到系统托盘，右键菜单快速操作。

安装

git clone https://github.com/QingdaoDitie/QuickDock.git
cd QuickDock/QuickDock
dotnet build -c Release

编译完成后，可执行文件位于 bin/Release/net8.0-windows/QuickDock.exe

使用

1. 双击运行 QuickDock.exe

2. 右键点击托盘图标，选择"设置"

3. 在设置中添加快捷方式、配置城市、设置开机自启

4. 鼠标移至屏幕热区显示 Dock，移开后自动隐藏
